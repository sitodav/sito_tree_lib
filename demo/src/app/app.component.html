Sito-Tree demo

Sito-Tree is a rendering library for nested/hierarchical json data representation in trees/graph form.

The library allows interactive creation of a tree/graph, or creation from json data (it's configurable to allow agnostic data representation!)

There are several examples below to show some of the properties/behaviours and use cases for the library.

The most interesting part is at the end, where you can see how to read json data into a tree/graph, save and restore.


 Library last update : 10/09/2023

Example 1: Interactive, no autosize, color by cluster (nodes get the same color of the father/cluster).
Click on the canvas to create a new node
Drag a node on top of another (on a different cluster) to append to, double click a node to expand. Mouseover on edges highlights them, and the click select it.
While interacting, open the console to see the registered callbacks logging

Example 2: Custom graphical props. Readonly, autosize, color by state (the color of a node depends on its internal status).
The tree is non modifiable, and it's nodes are loaded asynchronously (after some second) from read (mocked) data

Double click a node to expand. Click update button to change node state/color   , or expand  

Example 3: Same as before (but clickable edges with callback for setting highlight and click handler on nodes for highlight on those too): trees are disposed along row-wise using a fixed layout.

Example 4: Same as before, but node are disposed using column-wise layout

Example 5: Tree loaded via json and interactive, with data exporting in json. You can configure what json schema to use and what data use as node status.
The schema tells the library how to interpret the json data and map it to trees/graph nodes.
Interact to create new nodes on the loaded (via json data) tree. On every interaction with the tree (create/append etc) the data is exported (on the right)
